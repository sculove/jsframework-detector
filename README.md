# jsframework-detector
[chrome extension] JS Framework detector

## Support JS Librarys/Frameworks
- Jindo
- Jindo Component
- Jindo Mobile Component
- jQuery
- jQuery Mobile
- Underscore
- Backbone
- Angularjs
- Requirejs
- React
- Knockout
- Ember
- Handlerbars
- Polymer
